Rangkuman

Komputer

Komputer adalah sebuah perangkat elektronik yang dikendalikan oleh instruksi user untuk mengolah atau menyimpan data. Bentuk dari computer juga beragam, seperti laptop, tablet, desktop, all-in-one, dan server. Kemudian ada pula mobile computer seperti smartphone, digital kamera, portable digital media player dan wearable device (smart watch).

Cara kerja sebuah komputer?

Perlu diketahui bahwa tugas computer ialah memproses sebuah data. Jadi halyang dilakukan sebagai user ialah memberikan sebuah data (input) barulah sebuah computer bisa mengolah data tersebut (processing) dan memproduksi hasil dari proses (output).

input dan Output pada komputer

Input device ialah device yang digunakan untuk memberikan sebuah perintah kepada computer seperti: pointing device (mouse), keyboard, mic, dan touchpad pada laptop.

Output device ialah perangkat keras yang mengubah data yang sudah di proses oleh computer, biasanya beriupa text, grafik, audio, video, dan benda(3D print).

Sistem penyimpanan pada komputer

Ketika sudah mengolah data, perlulah sebuah computer itu menyimpan data tersebut di storage device. Storage device pada computer bermacam jenisnya seperti yang paling umum digunakan pada saat ini ialah hard disk(HDD), ada pula solid-state drive(SSD), USB flash drive, memory card, optical drive, dan cloud storage. Fungsi dan cara kerja dari cloud storage ini hamper sama dengan storage device lainnya, hanya saja cloud storage ini memrlukan internet untuk mengaksesnya.

Dalam penggunaan computer ada istilahGreen computing ialah gerakan untuk menguramgi sumber listrik yang digunakan seperti recycling, regulasi manufaktur, memperlama masa hidup dari suatu komponen computer, dan menggunakan efficiency energy pada hardware baru computer.

Dalam menjalankan sebuah computer perlulah sebuah system operasi dan beberapa software untuk bisa menjalankan sebuah computer dengan lancer.

internet

Dalam mengakses internet ada yang bernama World Wide Web (WWW/web) ialah sebuah Pustaka global yang terkoneksi dan diakses melalui internet, biasanya dalam World Wide Web (WWW) ini digunakan untuk mengakses sebuah webpage yang berada di internet seperti webpage www.wikipedia.org . Bagian dari webpage bernama website, dan webpage ini haruslah mempunyai system web server agar bisa di akses oleh semua orang yang berada di internet. Dalam upaya mengakses internet kita perlu mempunyai sebuah browser yang mempunyai search engine yang andal dalam computer kita agar bisa mengakses internet dengan lancer.

Komunikasi dan Networking

Pada abad 21 sebuah komunikasi dimudahkan dengan adanya internet hanya dengan koneksi internet dan beberapa perangkat komunikasi. Jaringan diabgi menjadi dua yaitu menggunakan kabel dan wireless (tanpa kabel) contoh dari wireless koneksi ialah wi-fi, Bluetooth, dan cellular radio.
